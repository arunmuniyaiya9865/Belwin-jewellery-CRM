# Phase 2: Design System

## 1. Typography
Using **Inter** (Primary) and **Outfit** (Headings) for a premium feel.

- **Heading XL:** 36px / Bold / #111827
- **Heading LG:** 30px / SemiBold / #111827
- **Heading MD:** 24px / SemiBold / #1F2937
- **Heading SM:** 18px / Medium / #1F2937
- **Body LG:** 16px / Regular / #374151
- **Body MD:** 14px / Regular / #4B5563
- **Body SM:** 12px / Regular / #6B7280

## 2. Color Palette
A premium palette using Gold and Sleek Dark tones.

| Name | Hex Code | Usage |
| :--- | :--- | :--- |
| **Primary (Gold)** | #B8860B | Brand identity, primary buttons |
| **Secondary (Slate)** | #0F172A | Sidebar, backgrounds |
| **Success** | #10B981 | Completed payments, growth |
| **Warning** | #F59E0B | Pending installments, low stock |
| **Danger** | #EF4444 | Overdue, stock out, errors |
| **Neutral** | #F3F4F6 | Surface backgrounds |

## 3. Core Components
All components will be built with Tailwind CSS and Framer Motion for smooth interactions.

- **Buttons:** Solid, Outline, Ghost (with hover scaling).
- **Cards:** White surface with subtle shadow and border.
- **Tables:** High-density, filterable, with status badges.
- **Modals:** Centered overlays with backdrop blur.
- **Inputs:** Floating labels, validation states.

## 4. Iconography
Using **Lucide React** for clean, consistent outlines.
